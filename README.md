# npm-boilerplate

해당 프로젝트는 `create-react-app --template typescript` 로 만들어졌습니다.

# 사용 기술

- `create-react-app`
- `typescript`
- `emotion`

# 사용 방법

1. 해당 프로젝트 `Clone` 하기

2. `package.json`의 `name`을 변경해주세요.

3. 개발은 `src/lib`에서 진행하면 됩니다.

4. `npm publish` 명령어를 통해서 npm 배포를 진행해주세요.
   - `npm login`이 되어있어야 합니다.
